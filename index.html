newebe

Get the data back home

(Almost) Distributed social platform

Contact

http://groups.google.com/group/newebe

Newebe is...

  • a social tool to share data with contacts or friends
  • a web application
  • self-hosted
  • mono-user

Newebe is not...

  • a web application to be hosted as a public website
  • a Jabber server
  • a Facebook like

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/gelnior/newebe
